# Telegram RAG‑бот с Groq и Hugging Face эмбеддингами

## Требования
- Python 3.11
- Переменные окружения: TELEGRAM_TOKEN, GROQ_API_KEY, HF_TOKEN
- HTML‑документы в папке `/app/data/docs` (на хостинге)

## Установка
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите бота: `python bot.py`
