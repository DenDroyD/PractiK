# RAG-бот для Telegram с использованием Groq, FastEmbed и ChromaDB

Бот отвечает на вопросы, используя ваши HTML-документы.

## Настройка

1. Добавьте переменные окружения в панели управления bothost.ru:
   - `TELEGRAM_TOKEN` – токен вашего Telegram-бота
   - `GROQ_API_KEY` – ключ API Groq

2. Загрузите HTML-файлы в папку `/app/data/docs` через файловый менеджер.

3. После деплоя бот автоматически проиндексирует документы и начнёт отвечать.

## Зависимости

Указаны в `requirements.txt`. Используются лёгкие компоненты (без PyTorch), что позволяет работать на тарифе Basic (1 ГБ RAM, 5 ГБ диска).
